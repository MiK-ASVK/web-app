Variables and Data Types in Python

In Python, a variable is a name that refers to a value stored in the computer's memory. Variables are used to store data values, and their values can change during the execution of a program.

Unlike some other programming languages, Python variables do not need to be explicitly declared with a specific data type. Instead, the type of a variable is inferred from the value assigned to it.

Variables names

A variable can have a short name (like x and y) or a more descriptive name (age, carname, total_volume). Rules for Python variables:

  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive (age, Age and AGE are three different variables)
  • A variable name cannot be any of the Python keywords.

Global Variables

Variables that are created outside of a function (as in all of the examples above) are known as global variables. Global variables can be used by everyone, both inside of functions and outside.

If you create a variable with the same name inside a function, this variable will be local, and can only be used inside the function. The global variable with the same name will remain as it was, global and with the original value.

Example of Variable Assignment:

# Assigning values to variables
x = 5             # Integer variable
y = "Hello, World!"  # String variable
z = 3.14          # Float variable

Variables do not need to be declared with any particular type and can even change type after they have been set. 

x = 4       # x is of type int
x = "Sally" # x is now of type str

Python has various built-in data types that represent different kinds of values that can be stored and manipulated in a program. Some of the commonly used data types include:

  • int: Integer values, e.g., 5, -3, 100.
  • float: Floating-point values, e.g., 3.14, -0.001, 2.5e-5.
  • str: String values, e.g., "Hello", 'Python', "123".
  • list: Ordered collection of items, e.g., [1, 2, 3], ['a', 'b', 'c'].
  • tuple: Ordered, immutable collection of items, e.g., (1, 2, 3), ('a', 'b', 'c').
  • dict: Collection of key-value pairs, e.g., {'name': 'John', 'age': 30}.
  • set: Unordered collection of unique items, e.g., {1, 2, 3}, {'a', 'b', 'c'}.
  • bool: Represents True or False.

Each of these data types serves a different purpose and can be used in various operations.
