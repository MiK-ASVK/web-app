Control Flows in Python

Control flow statements in Python allow you to control the execution of your program's code. They help you to make decisions, repeat actions, and execute code conditionally. The main types of control flow statements in Python are:

  • Comparison Operators
  • If Statements
  • For Loops
  • While Loops

Comparison Operators

Comparison operators are used to compare two values. They are essential in control flow statements like if statements. The comparison operators in Python are:

  • == : Equal to
  • != : Not equal to
  • > : Greater than
  • >= : Greater than or equal to
  • < : Less than
  • <= : Less than or equal to

Examples:

x = 5
y = 10

# Equal to
print(x == y)  # Output: False

# Not equal to
print(x != y)  # Output: True

# Greater than
print(x > y)   # Output: False

# Greater than or equal to
print(x >= y)  # Output: False

# Less than
print(x < y)   # Output: True

# Less than or equal to
print(x <= y)  # Output: True

If Statements

The if statement is used to test a condition. If the condition is true, the block of code inside the if statement is executed. Optionally, you can add elif and else statements to handle multiple conditions.

age = 18

if age >= 18:
    print("You are an adult.")
elif age > 12:
    print("You are a teenager.")
else:
    print("You are a child.")

In this example, the program will output You are an adult. because the condition age >= 18 is true.

For Loops

The for loop is used to iterate over a sequence (such as a list, tuple, dictionary, set, or string) and execute a block of code for each item in the sequence.

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

This loop will print each fruit in the fruits list:

  • apple
  • banana
  • cherry

While Loops

The while loop is used to repeatedly execute a block of code as long as a condition is true.

count = 0

while count < 5:
    print("The count is", count)
    count += 1

This loop will print the value of count from 0 to 4:

  • The count is 0
  • The count is 1
  • The count is 2
  • The count is 3
  • The count is 4

Conclusion

Control flow statements are essential for writing effective and efficient programs in Python. By using comparison operators, if statements, for loops, and while loops, you can control the flow of your program and handle different scenarios and repetitive tasks with ease.
